# 社区行为准则

天空农云欢迎不同专业、经验和立场的人参与。我们鼓励大胆想象，也要求对证据、风险和彼此保持尊重。

## 我们承诺

- 讨论观点和证据，不攻击个人。
- 欢迎反例、失败记录和“不确定”。
- 尊重专业边界，不把推测包装成结论。
- 让新参与者能够理解术语、假设和下一步。
- 不容忍骚扰、歧视、威胁、泄露隐私或持续破坏性行为。

## 维护者责任

维护者可以编辑、隐藏或删除违反本准则的内容，并在必要时限制参与。处理过程应以保护社区安全和研究完整性为目标。

## 报告问题

在公开报告会伤害当事人或泄露隐私时，请通过仓库维护者的 GitHub 个人资料联系维护者。报告会被谨慎处理。

本准则参考 Contributor Covenant 的核心原则，并针对开放工程研究社区作了简化。
