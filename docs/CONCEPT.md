# 技术概念与研究边界

## 1. 概念

天空农云把一组轻质、模块化、可回收的系留浮空单元布置在非核心城市场地上方。每个单元尝试协调四种用途：

1. 为选定作物保留适宜光谱和光周期；
2. 通过作物蒸腾与受控水循环带走热量；
3. 回收可用雨水、冷凝水和低品位热；
4. 为传感、通信、姿态控制和安全回收提供必要电力。

这不是“太阳能板加温室”的简单叠加，也不假设所有波段都应转化为电。核心研究对象是光、水、热、重量、风载和公共空间之间的系统分配。

## 2. 已有证据能说明什么

- 植被通过遮阴和蒸散影响城市表面能量平衡；NASA 的城市热岛资料把植被缺失、蒸散减少和表面升温联系起来。
- 农光互补研究显示，遮阴效果高度依赖作物、气候、阵列几何和年份。NREL/NLR 的多年田间结果中，不同年份的可销售产量表现不同，因此不能把单次“增产”外推为普遍规律。
- FAO 的作物蒸散框架表明，作物耗水需要由天气需求、作物系数、生长阶段和有效降水共同估算。
- 系留气球属于受监管的空域对象；例如美国 FAA 对系留气球、系留缆标志、通知和运行条件有专门规则。其他国家和城市必须使用当地法规。

## 3. 第一性量级

面积为 `A` 平方米、辐照度为 `G` W/m² 时，几何截获太阳功率为：

`P = A × G`

这只是入射功率，不是发电量、作物有效辐射或可实现降温量。至少还要扣除或区分：

- 膜材反射、吸收与透射
- 光谱选择性
- 平台投影和太阳高度角
- 云量与气溶胶
- 作物光合饱和和光周期
- 泵送、控制和热交换损耗
- 风致变形和安全收拢时间

蒸腾冷却也不是“免费制冷”。水的汽化潜热能提供可观冷却，但水源、泵送、盐分、营养液、病害和回收率都可能成为主约束。

## 4. 优先研究问题

### 作物与光谱

- 哪些高价值、低水库存作物适合动态部分遮阴？
- 选择性膜能否在不降低关键作物品质的前提下减少过量辐射？
- 云幕移动造成的光斑和光周期波动如何影响作物？

### 结构与回收

- 单元面积多大时，风载、缆绳和部署复杂度的总成本最低？
- 哪种失效方式最安全：泄压、分离、降落还是地面回卷？
- 水库存、营养液和维护通道对净升力的影响有多大？

### 城市气候

- 阴影应覆盖道路、停车场、屋顶还是公共空间？
- 地表降温是否转化为空气温度与建筑冷负荷下降？
- 不同湿度条件下，蒸腾冷却的水代价与热舒适收益如何变化？

### 治理

- 谁决定阴影的时间、位置和受益者？
- 如何处理居民采光、屋顶光伏、航空、鸟类和景观影响？
- 试点失败、退出和恢复地面原状的责任如何分配？

## 5. 否决条件

出现以下任一情况，应停止扩大规模并重新设计：

- 在目标天气窗口内无法可靠回收；
- 单点失效可能进入人口密集区或关键空域；
- 水、材料或氦气全生命周期影响抵消主要环境收益；
- 作物或微气候收益只能在不可复制的条件下出现；
- 阴影分配无法形成透明、公平、可申诉的治理机制。
